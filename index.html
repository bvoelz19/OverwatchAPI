OverwatchAPI

A RESTful API that serves Overwatch player data as JSON on request


Usage details

HTTP GET

https://overwatchapi.herokuapp.com/query?platform=YOUR_PLATFORM&username=YOUR_USERNAME
  • Supported platforms: "PSN", "PC", "XBOX"
  • Username is case-sensitive

Example

Command: curl https://overwatchapi.herokuapp.com/query?platform=PSN&username=bvoelz19
